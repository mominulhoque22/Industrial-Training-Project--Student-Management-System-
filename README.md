# Student Management System

A **Student Management System** developed as part of my **Industrial Training Project**, where I was responsible for the **backend development** using **Django**. The system is designed to efficiently manage student information for the **Science Faculty**.

The project supports multiple departments, including:

* Computer Science & Engineering (CSE)
* Electrical & Electronic Engineering (EEE)
* Computer & Communication Engineering (CCE)
* Electronics & Telecommunication Engineering (CCE)
* Electronics & Telecommunication Engineering (ETE)
* Civil Engineering (Civil)

## Features

* Student Registration and Management
* Add, Update, Delete, and View Student Records (CRUD)
* Department-wise Student Management
* Secure Backend with Django
* Database Integration
* Organized and Scalable Architecture

## Technologies Used

* Python
* Django
* MySQL
* REST API (if applicable)
* HTML, CSS, JavaScript
* Bootstrap

## My Contribution

* Developed the complete backend using **Django**.
* Designed and implemented database models.
* Built CRUD functionalities for student management.
* Managed department-wise student data.
* Implemented backend logic and database connectivity.

## Project Objective

The objective of this project is to provide an efficient and centralized platform for managing student records across different departments within the Science Faculty. It also demonstrates practical experience in backend development using Django and database management gained during industrial training.
